# Result Publishing System

This project contains two Java-based Result Publishing Systems:

1. Swing GUI Result Publishing System
2. Console-Based Result Publishing System

---

# Project Structure

```text
Repository
│
├── Swing_Result_System
│    ├── MainApp.java
│    ├── ResultGUI.java
│    ├── ResultService.java
│    └── Student.java
│
└── Console_Result_System
     └── MarksCard
          └── Student.java
```

---

# Software Requirements

- Java JDK 8 or above
- VS Code / Eclipse IDE (optional)

---

# Step 1 : Check Java Installation

Open terminal or command prompt and run:

```bash
java -version
javac -version
```

If Java is installed correctly, version details will appear.

---

# Step 2 : Download the Project

Download or clone the repository from GitHub.

Extract the ZIP file if downloaded as ZIP.

---

# Step 3 : Run Swing GUI Result System

## Open Terminal Inside:

```text
Swing_Result_System
```

---

## Compile Java Files

```bash
javac *.java
```

---

## Run the Program

```bash
java ResultGUI
```

The Swing GUI application window will open.

---

# Step 4 : Run Console Result System

## Open Terminal Inside:

```text
Console_Result_System
```

---

## Compile Java File

```bash
javac MarksCard/Student.java
```

---

## Run the Program

```bash
java MarksCard.Student
```

Console output showing student results will appear.
